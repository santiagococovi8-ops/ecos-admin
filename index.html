🗄️ Datos iniciales

Si Supabase está vacío, usá este botón para cargar todos los datos iniciales de la app (noticias, apuntes y ECódigos). Solo es necesario hacerlo una vez. Si ya tenés datos, no hace nada.

📢 Enviar notificación a todos los usuarios

Se envía a todos los dispositivos suscritos. Requiere que el usuario haya aceptado los permisos de notificaciones en la app.

👤 Enviar notificación a usuario específico

Se envía solo al dispositivo del alumno indicado, usando su legajo como identificador. El alumno debe haber iniciado sesión en la app al menos una vez.

🔍 Diagnóstico del sistema de notificaciones

Verificá el estado de cada componente del sistema: endpoint Vercel, OneSignal, tabla Supabase y suscriptores activos.

Presioná "Ejecutar diagnóstico" para ver el estado del sistema.

  id           UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  titulo       TEXT NOT NULL,
  mensaje      TEXT NOT NULL,
  tipo         TEXT DEFAULT 'push',
  emoji        TEXT DEFAULT '🔔',
  destinatario TEXT DEFAULT 'todos',
  legajo       TEXT,
  fecha        TIMESTAMPTZ DEFAULT NOW(),
  fecha_display TEXT,
  activa       BOOLEAN DEFAULT TRUE,
  onesignal_id TEXT,
  recipients   INTEGER DEFAULT 0
);
